== Description ==
Woocommerce Hide Options plugin very simple options for hide meta and other options from Product single page like: Category, Tags, comments, Price, Star Review, Cart Button and other many thinks from Single product page.
Hide Coupon Code form from cart page and Checkout page. 

== Installation ==
<strong>To install "Woocommerce Hide Options" using the built-in plugin installer: </strong>
1. Go to Plugins > Add New.    
2. Type in "Woocommerce Hide Options" in Search Plugins box or click a tag link below the screen.    
3. find the "Woocommerce Hide Options" in the list.    
4. Click Install Now to install the WordPress Plugin.    
5. Go to Woocommerce menu in wp-admin area.    
6. Select Hide Option Link.
7. Choose  option what you want to Hide from Product details page

<strong>To install "Woocommerce Hide Options" manually:  </strong>
1. Download your WordPress Plugin to your desktop.   
2. If downloaded as a zip archive, extract the Plugin folder to your desktop.   
3. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.   
4. Go to Plugins Menu and find the "Woocommerce Hide Options" in the list.   
5. Click on Activate button to activate it.   
6. Go to Woocommerce menu in wp-admin area.   
7. Select Hide Option Link.  
8. Choose  option what you want to Hide from Product details page
